    • Флажок "Повторно"



    Установите этот флажок, если требуется
    печатать повторное свидетельство.




    Дата выдачи свидетельства



    По умолчанию ЭРАГС предлагает текущую системную дату.
    Если необходимо, укажите другую дату выдачи свидетельства.




    Флажок "Как дата регистрации"



    Установите этот флажок, если нужно, чтобы дата выдачи
    свидетельства совпадала с датой регистрации.




    Список подчинённых отделов



    По умолчанию в качестве места регистрации ЭРАГС предлагает
    отдел, который соответствует установкам. Если нужно, чтобы в
    качестве места регистрации был указан другой отдел,
    выберите этот отдел из списка.




    Место регистрации



    По умолчанию в качестве места регистрации ЭРАГС предлагает
    отдел, который указан в установках. Если нужно, чтобы в
    качестве места регистрации был указан другой отдел,
    выберите этот отдел из списка подчинённых отделов.
    Отредактируйте место регистрации, если это необходимо.
    ЭРАГС сохраняет место регистрации отдельно для каждого
    элемента списка и для каждого вида регистрации.

    Если нужно вставить строку, нажмите Ctrl+Enter.




    Орган, выдавший свидетельство



    По умолчанию ЭРАГС предлагает отдел, который указан
    в установках. Отредактируйте, если это необходимо.
    ЭРАГС сохраняет значение этого поля отдельно для
    каждого вида регистрации.

    Если нужно вставить строку, нажмите Ctrl+Enter.




    Начальник отдела



    Значение этого поля будет напечатано в свидетельстве,
    если в шаблоне есть соответствующее ключевое слово.
    По умолчанию ЭРАГС предлагает значение из установок.
    Отредактируйте значение поля, если это необходимо.
    ЭРАГС сохранит значение этого поля до конца сеанса работы.
    Если нужно изменить ФИО начальника по умолчанию,
    откройте вкладку "Загальні установи" из диалогового окна
    "Установи" и отредактируйте поле "ПІБ керівника відділу".




    Кнопка "Так"



    Нажмите эту кнопку, чтобы ЭРАГС сформировал документ,
    для печати и вызвал его для предварительного просмотра.




    Кнопка "Ні"



    Нажмите эту кнопку, если Вы не собираетесь,
    просмотреть или напечатать свидетельство.

    Дата решения суда



    Введите дату решения суда об усыновлении.
    Дата должна быть правильной и определена полностью.
    Формат даты: дд.мм.гг или дд.мм.гггг:
    · по две цифры на число и месяц и четыри или две цифры на год;
    · две цифры года можно вводить, если год больше 1929.




    Номер решения суда



    Введите номер решения суда об усыновлении.
    Допускается вводить любые символы.
