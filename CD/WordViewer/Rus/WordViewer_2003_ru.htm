Просматривайте, печатайте и копируйте документы Word, даже если приложение Word не установлено. Этот загружаемый файл заменяет средство просмотра Word 97 и все предыдущие версии этого средства.

Обзор

Word Viewer 2003 позволяет открывать документы Word 2003 и документы, созданные во всех предыдущих версиях Microsoft Word для Windows® и Microsoft Word для Macintosh. В дополнение к документам Word (файлам с расширением DOC) можно также открывать файлы в следующих форматах:

  • Rich Text Format (RTF)

  • текстовые файлы (TXT)

  • форматы веб-страниц (HTM, HTML, MHT, MHTML)

  • WordPerfect 5.x (WPD)

  • WordPerfect 6.x (DOC, WPD)

  • Works 6.0 (WPS)

  • Works 7.0 (WPS)

  • Extensible Markup Language (XML)


С помощью средства просмотра Word 2003 можно просматривать, печатать и копировать содержимое документов в другую программу. При этом невозможно редактировать открытый документ, сохранять его или создавать новый документ. Этот загружаемый файл заменяет средство просмотра Word 97 и все предыдущие версии этого средства.

  • Поддерживаемые операционные системы: Windows 2000 Service Pack 3; Windows Server 2003; Windows XP

    • Microsoft Word 2002
    • Microsoft Word 2000

Инструкции

Для установки данного средства:
  1. Нажмите кнопку Загрузить в верхней части страницы и сохраните загружаемый файл на жестком диске.
  2. Дважды щелкните файл wdviewer.exe на жестком диске, чтобы запустить программу установки.
  3. Выполните установку, следуя указаниям на экране.
Указания по применению

После выполнения установки откройте файл с помощью средства просмотра Word 2003.
  1. В меню Файл выберите команду Открыть.
  2. В окне Открытие документа выберите дисковод, папку или адрес в Интернете, где находится файл, который нужно открыть.
  3. Найдите в папке нужный файл.
  4. Выберите этот файл, затем нажмите кнопку Открыть.
Примечание. Если необходимо просмотреть содержимое файла до нажатия кнопки Открыть, в меню Вид выберите пункт Предварительный просмотр.

Как удалить данный загружаемый файл:

Для удаления загружаемого файла следует удалить программу wdviewer.exe.

  1. В меню Пуск выберите пункт Настройка и откройте Панель управления.
  2. Дважды щелкните значок Установка и удаление программ.
  3. В списке установленных программ выберите Microsoft Office Word Viewer 2003 и нажмите кнопку Удалить или Заменить/Удалить. При появлении диалогового окна следуйте указаниям для удаления программы.
  4. Чтобы подтвердить необходимость удаления, нажмите Да или ОК.
