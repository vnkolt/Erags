Списки

Програма Erags для введення даних використовує такі списки:

  • географічних назв
  • органів реєстрації
  • громадянств
  • вулиць
  • органів внутрішніх справ
  • пологових будинків, лікарень, моргів
  • фінансових установ
  • імен
  • по батькові
  • причин смерті

    • Списки потрібні для швидкого і зручного введення даних. Списки географічних назв и органів реєстрації мають деревоподібну структуру, яка відображає логічні зв'язки відповідних даних. Для роботи з цими списками у програмі Erags передбачені спеціальні вікна, які відображають деревоподібну структуру цих списків.

    Списки пологових будинків, лікарень, моргів і причин смерті мають так звані псевдоніми. Використання псевдоніма дозволяє запровадити введення досить довгої назви за допомогою кількох натискань клавіш. Приклад: для введення причини смерті "Ішемічна хвороба серця" достатньо ввести псевдонім "іхс", а для введення назви лікарні "Донецьким областним лечебно-клиничним об'єдненням" достатньо ввести псевдонім "ДОЛКО".

    Список географічних назв

    Список географічних назв використовується програмою Erags там, де потрібно визначити географічну назву. Список має деревоподібну структуру. Порядок введення географічних назв завжди однаковий: країна, область, район, місто (смт, село). Розглянемо введення групи географічних назв Україна, Вінницька, Барський, Діброва слщ. Назва Україна пропонується автоматично, під час переходу на поле "Область" буде запропонована назва, яка вказана в установах, список областей буде наповнений тільки областями України і для заповнення поля "Область" значенням Вінницька достатньо ввести "ві". Коли поле "Держава" заповнено значенням Україна, а поле "Область" - значенням Вінницька, список районів буде заповнений тільки районами Вінницкої області і для введення значення Барський у поле "Район" достатньо ввести лише літеру "Б". При цьому список населених пунктів буде заповнено тільки населеними пунктами Барского района Вінницкої області і для введення значения Діброва слщ в поле "м.с.") достатньо лише ввести літеру "Д".

    Щоб переглянути або відредагувати список географічних назв натисніть комбінацію Ctrl+U чи кнопку на головній панелі інструментів. На малюнку нижче показано вікно зі списком географічних назв з розкритими вузлами списку.


    Для навігації по списку використувуйте мишу або клавіши управління курсором. Для редагування елементів списка скористайтеся контекстним меню. Для виклику контекстного меню натисніть праву клавішу миші на якомусь елементі списку. На малюнку нижче показано контекстне меню списку:

    Список органів реєстрації

    Список органів реєстрації використовується програмою Erags там, де потрібна запровадити орган реєстрації, а також для друку конвертів. Список має деревоподібну структуру. На малюнку нижче показано вікно зі списком органів реєстрації.


    Не всі відділи зі списку органів реєстрації потраплятимуть у поля зі списками. За умовчанням відповідне поле зі списком буде наповнюватись лише відділами РАЦС, які належать до відділу, визначеному в установах як вищестоячий відділ. Якщо Ви бажаєте, щоб у поля зі списком потрапляли відділи РАЦС з іншої області, скористайтесь контекстним меню. Для цього оберіть потрібний відділ, натисніть праву клавішу миші і клацніть на позиції "Додавати до поля зі списком":


    Список органів реєстрації можна використовувати як довідник відділів РАЦС, Ви можете переглядати і відредагувати інформацію про відділ: адресу, телефони, примітки. Крім того, використовуючи список органів РАЦС, Ви можете друкувати конверти. Докладніше про друк конвертів дивіться в розділі Друк конвертів.

    Список громадянств

    Список громадянств використовується програмою Erags там, де потрібно ввести громадянство. Список має дві колонки: "Коротка Назва" і "Довга Назва". Під час введення громадянства слід вводити коротку назву. Для генерації документа Erags використовуватиме довгу назву. Приклад: якщо у поле "Громадянство" Ви вписали "Ізраілю", то для генерації актового запису у графі "Громадянство" Erags впише "Держави Ізраіль". Для перегляду або редагування списку громадянств оберіть його з меню "Списки". На малюнку нижче показане вікно зі списком громадянств.


    Список вулиць

    Список вулиць використовується програмою Erags там, де потрібно ввести назву вулиці. Для зручності введення рекомендується спочатку експлуатації ПК ЕРАГС наповнити список вулиць назвами, які найчастіше зустрічаються у Вашому відділі РАЦС. Для перегляду або редагування списку вулиць оберіть його з меню "Списки". На малюнку нижче показано вікно з прикладом списку вулиць.


    Список органів внутрішніх справ

    Список органів внутрішніх справ використовується програмою Erags там, де потрібно ввести назву органу, який видав паспорт. За умовчанням програма Erags пропонує той орган внутрішніх справ, який визначено в установах. Для перегляду або редагування списку органів внутрішніх оберіть його з меню "Списки". На малюнку нижче показано вікно з прикладом списку органів внутрішніх справ.


    Список пологових будинків, лікарень, моргів

    Список пологових будинків, лікарень, моргів використовується програмою Erags під час реєстрації смерті (орган, який видав медичне свідоцтво смерті) і під час реєстрації народження (орган, який видав медичне свідоцтво про народженні). Для зручності введення рекомендується на початку експлуатації ПК ЕРАГС наповнити цей список назвами відповідних установ, які найчастіше зустрічаються у Вашому відділі РАЦС. Список має дві колононки: "Псевдонім" і "Назва". Визначати псевдоніми можна як Ваш заманеться, проте краще, якщо псевдонім асоціюється із відповідною назвою. Використання псевдонімів дозволяє вводити довгі назви швидше. Для перегляду або редагування списку пологових будинків, лікарень і моргів слід вибрати його з меню "Списки". На малюнку нижче показано вікно з прикладом списку пологових будинків, лікарень і моргів.


    Список фінансових установ

    Список фінансових установ використовується програмою Erags при реєстрації розірвання шлюбів і зміни імені. Для зручності введення рекомендується на початку експлуатації ПК ЕРАГС запровадити кілька назв фінансовий установ, які найчастіше зустрічаються в Вашем відділі РАЦС, тобто розташовані неподалік Вашого відділу РАЦС. Зазвичай досить двох-трьох назв, щоб забезпечити швидке введення назви фінансової установи до відповідного поля. Програма Erags дозволяє ввести до списку фінансових установ до 20-ти назв. Для перегляду або редагування списку фінансових установ слід оберіть його з меню "Списки". На малюнку нижче показано вікно для перегляду і редагування з прикладом списку фінансових установ.


    Список імен

    Список імен використовується програмами Erags и EraABC всякий раз, коли треба ввести ім'я до відповідного поля. Кожне значення списку імен має ознаку "Стать". Щоб змінити цю ознаку, наведіть курсор миші на відповідну чарунку списку і клацніть лівою клавішею миші. Щоб переглянути або відредагувати список імен оберіть його з меню "Списки". На малюнку нижче показано вікно для перегляду або редагування списку імен.


    Список отчеств

    Список по батькові використовується програмами Erags і EraABC щоразу, коли треба ввести по батькові до відповідного поля. Кожне значення списку по батькові має ознаку "Стать". Щоб змінити цю ознаку, наведіть курсор миші на відповідну чарунку списку і клацніть лівою клавішею миші. Щоб переглянути або відредагувати список по батькові оберіть його з меню "Списки". На малюнку нижче показано вікно для перегляду або редагування списку по батькові.


    Список причин смерті

    Список причин смерті використовується програмою Erags для введення причини смерті під час реєстрації смерті. Список має дві колонки: "Псевдонім" і "Причина смерті". Використання псевдонімів дозволяє швидко ввести причину смерті. Приклади: введення псевдоніма "аспмкпн" визначить причину смерті "Атеросклеротичний порок мітрального клапану із перевагою недостатності.", а введення псевдоніма "іхсгкнн" визначить причину смерті "Ішемічна хвороба серця: гостра коронарна недостатність.". Перед експлуатацією ПК ЕРАГС рекомендується ознайомитися зі списком причин смерті, щоб потім спростити його використання для введення псевдонімів. Для перегляду або редагування списку причин смерті оберіть його з меню "Списки". На малюнку нижче показано вікно з переліком причин смерті.
