Главное меню

Главное меню - это полоска с текстом в верхней части главного окна программы. С помощью элементов главного меню вы можете открывать и создавать файлы базы данных ЭРАГС, выбирать нужный вид регистрации, осуществлять навигацию по базе данных и выполнять множество других операций.

При первом запуске программы Erags не все элементы меню и панелей управления доступны. Это связано с тем, что не создано ни одного файла базы данных.

Для начала работы с ПК ЭРАГС нужно создать файл базы данных. Для этого необходимо воспользоваться командой меню Файл/Новий или соответствующей кнопкой основной панели инструментов.

После выбора команды меню Файл/Новий появится диалоговое окно для определения имени файла базы данных.



Каждый файл базы данных ЭРАГС является электронным аналогом книг регистрации актовых записей за соответствующий год. Поэтому во избежании путаницы важно, чтобы в начале имени файла стояли четыре цифры - год, к которому относится этот файл.

Перед началом работы следует проверить установки и, если необходимо, исправить значения установок. Для этого нужно воспользоваться командой меню Огляд/Установи.

После выбора команды меню Огляд/Установи появится диалоговое окно Установи:

Обратите внимание, что после названия населенного пункта нужно указать:

  • м. - для города (місто);
  • с. - для села;
  • с-ще або слщ - для посёлка (селище);
  • смт - для поселка городского типа (селище міського типу);
  • х - для хутора
При указании названия отделов РАГС необходимо использовать аббревиатуры:
  • ВРАГС - відділ реєстрації актів цивільного стану;
  • ВРАЦС - відділ реєстрації актів цивільного стану;
  • ОУЮ - обласного управління юстиції;
  • МУЮ - міського управління юстиції;
  • РУЮ - районного управління юстиції;
  • МРУЮ - міськрайонного управління юстиції;
  • ГУЮ - Головного управління юстиції
  • В - Виконком;
  • о - області;
  • р - району;
  • мр - міської ради;
  • ср - сільської ради;
  • сщр - селищної ради

Перед началом работы с ПК ЭРАГС полезно также ознакомиться со списками, которые будут использоваться для быстрого ввода данных. Чтобы вызвать списки для просмотра/редактирования, необходимо воспользоваться командой меню Списки и открыть нужную таблицу.

Наиболее зависимым к конкретному отделу РАГС является список медицинских учреждений (Пологові будинки, лікарні, морги). Поэтому перед началом работы следует открыть этот список и указать названия и псевдонимы тех медицинских учреждений, которые наиболее часто используются для регистрации смертей и рождений в вашем отделе РАГС.
