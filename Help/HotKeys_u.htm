"Гарячі" клавіші

Для ефективної роботи з ПК ЕРАГС зручно використовувати так звані "гарячі" клавіші. У таблиці нижче наведений список клавіш і комбінацій клавіш, використання яких заощадить час. Комбінації клавіш, зазначені червоним кольором дійсні тільки для програми Erags - основного компоненту ПК ЕРАГС.

Комбінація клавіш Дія
Ctrl+1 Відкриває екранну форму для реєстрації народження.
Ctrl+2 Відкриває екранну форму для реєстрації шлюбу.
Ctrl+3 Відкриває екранну форму для реєстрації розірвання шлюбу.
Ctrl+4 Відкриває екранну форму для реєстрації зміни.
Ctrl+5 Відкриває екранну форму для реєстрації смерті.
Ctrl+U(укр. Г) Відкриває екранну форму списку географічних назв.
Ctrl+H(укр. Р) Відкриває екранну форму списку органів реєстрації.
Ctrl+N Викликає діалогове вікно для створення нового файлу бази даних (команда New).
Ctrl+O Викликає діалогове вікно для відкриття файлу бази даних (команда Open).
Ctrl+P(Print) У режимі екранної форми викликає меню друку документів. У режимі перегляду документа викликає діалогове вікно друку документу (команда Print).
Alt+Insert Створює новий запис.
Alt+Delete Віддаляє поточний запис.
Alt+Home Команда навігації: перехід на перший запис.
Alt+з Команда навігації: перехід на попередній запис.
Alt+и Команда навігації: перехід на наступний запис.
Alt+End Команда навігації: перехід на останній запис.
Ctrl+и Переводить каретку на наступне слово.
Ctrl+з Переводить каретку на попереднє слово.
Shift+Home Виділяє текст (або знімає виділення) від поточної позиції каретки до початку рядка.
Shift+End
Shift+и Виділяє символ (або знімає виділення) праворуч від каретки.
Shift+з Виділяє символ (або знімає виділення) ліворуч від каретки.
Ctrl+Shift+и Виділяє (знімає виділення) слово праворуч від каретки.
Ctrl+Shift+з Виділяє (знімає виділення) слово ліворуч від каретки.
Del Видаляє виділений текст або символ праворуч від каретки.
Backspace Видаляє виділений текст або символ ліворуч від каретки.
Ctrl+Insert Копіює виділений текст у буфер обміну (команда Copy).
Shift+Insert Заміщає виділений текст вмістом буферу обміну або уставляє вміст буфера обміну в поточне поле (команда Paste).
Shift+Del Вирізує виділений текст із поточного поля та поміщає його в буфер обміну (команда Cut).
Alt+в Розкриває список поточного поля зі списком.
Tab або Enter Перехід на наступний елемент введення.
Shift+Tab або Shift+Enter Перехід на попередній елемент введення.
Ctrl+Enter Уставляє новий рядок у поточну позицію каретки і переводить каретку на початок нового рядка. Тільки для текстового поля з декількома рядками, як, наприклад, поле приміток або причини смерті.

Комбінації клавіш треба використовувати в такий спосіб: утримуючи клавішу Ctrl, Shift, або Alt, натиснути на іншу клавішу комбінації.
