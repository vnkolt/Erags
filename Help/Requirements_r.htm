Требования к аппаратному обеспечению

Для эксплуатации программного комплекса ЭРАГС требуется компьютер Pentium (или совместимый с ним) с тактовой частотой процессора не менее 133 MHz (рекомендуется 633 MHz или более), памятью 32Mb (рекомендуется 128 Mb или более). Рекомендуется (но необязательно), чтобы к компьютеру был подключен манипулятор типа "мышь" или аналогичное устройство позиционирования курсора. Для полноценной работы с ПК ЭРАГС желательно, чтобы монитор и видеоадаптер поддерживали полноцветный (TrueColor) экранный режим с разрешением не менее 800x600 точек

Программа Erags позволяет печатать документы на принтеры любых типов, однако для эффективной работы следует использовать лазерный принтер. Большинство современных моделей лазерных принтеров обладают высокой скоростью и хорошим качеством печати.

Наиболее удобными принтерами для эксплуатации в отделах РАГС являются высокоскоростные современные лазерные принтеры с верхним лотком для подачи бумаги, как, например, принтер Canon LASER SHOT LBP-1120 или аналогичный.

Если в отделе РАГС имеется в наличии два компьютера, но лишь один принтер, то имеет смысл использовать принтер как общий для обоих компьютеров. Для этого необходимо связать оба компьютера в локальную сеть и открыть доступ к принтеру по сети. Для подключения компьютера к локальной сети необходимо, чтобы он имел сетевой адаптер.

Требования к программному обеспечению

На компьютере должна быть установлена какая-либо из операционных систем Windows 98, Windows Me, Windows NT 4.0, Windows 2000, Windows XP или Windows 7. Операционная система должна быть русскоязычной (украиноязычной) или русифицированной (украинизированной), то есть поддерживать кириллические шрифты и клавиатурный ввод украинским языком. На компьютере должна быть также установлена программа WinWord из пакета Microsoft Office 97 (Microsoft Office 2000, Microsoft Office XP, Microsoft Office 2003) или пакет OpenOffice.org.
