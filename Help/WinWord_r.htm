Настройка шаблонов

Любому печатному документу, который генерирует программа Erags, можно придать желаемый внешний вид. Для этого нужно изменить соответствующим образом шаблон документа. Необходимо помнить, что программа Erags самостоятельно не меняет шаблоны, она лишь использует их как образцы для генерации печатных документов. Наиболее удобно для настройки шаблонов использовать программу WinWord из пакета Microsoft Office. В данном разделе описаны некоторые приёмы и способы для эффективной настройки шаблонов с помощью программы WinWord из пакета Microsoft Offic 97.

Прежде, чем приступить к настройке шаблона в первый раз, убедитесь, что в программе WinWord установлены соответствующие флаги:

Параметры, вкладка Вид

  • графические объекты

  • границы области текста

  • пустые рамки рисунков



    • Параметры, вкладка Печать

  • скрытый текст

  • графические объекты



    • Сложность при изменении шаблонов может возникнуть, если ключевое слово находится внутри объекта, например, внутри объекта "надпись" или внутри ячейки таблицы. При настройке шаблонов может возникнуть необходимость переместить надпись, внутри которой находится ключевое слово или изменить выстоту и/или ширину ячеек таблицы.

    Перемещение объектов "надпись" и "рамка".

    Чтобы переместить объект, его нужно выделить (отметить). Нажмите и удерживайте клавишу Shift, а левой кнопкой мыши щёлкните на самом объекте. При этом станет видимой граница, окаймляющая объект. Если каретка включена, объект нельзя перемещать с помощью клавиш. Для того, чтобы отключить каретку, щелкните левой кнопкой мыши на границе, окаймляющей объект. После этого каретка изчезнет и объект можно перемещать с помощью клавиш перемешения (Влево, Вправо, Вверх, Вниз) - грубо, если при этом нажать и удерживать клавишу Ctrl - точно.

    Для группового перемещения объектов, нужно, удерживая клавишу Shift щёлкать левой кнопкой мыши на объектах, которые необходимо выделить для группового перемещения.

    Для изменения ширины и/или высоты объектов "надпись" или "рамка" нужно "схватить" мышкой за соответствующую границу и растянуть (сжать).

    Любое свойство объекта можно изменить, если вызвать окно свойств объекта. Для того, чтобы вызвать окно свойств, нажмите правую клавишу мыши на границе, окамляющей объект (вызов контекстного меню) и выберите "Формат надписи"

    Совместная работа Erags и WinWord во время настройки шаблонов.

    Не всегда получается настроить шаблон за один сеанс редактирования/проверки. Не обязательно закрывать WinWord для того, чтобы проверить результат настройки. Достаточно просто переключиться на программу Erags, выбрав ее на панели задач или с помощью комбинации клавиш Alt+Tab. Но перед тем, как проверять результат настройки, шаблон обязательно нужно сохранить (записать). Если документ, шаблон для которого Вы настраиваете, вызван для предварительного просмотра и Вы предполагаете продолжить настройку шаблона, то документ нужно сперва закрыть, а затем переключиться на WinWord.

    Проверка настройки шаблона свидетельства.

    Для проверки настройки шаблона свидетельства используйте листы бумаги по размеру бланка свидетельства. Откройте файл шаблона свидетельства в редакторе WinWord. Для этого нажмите на панели инструментов кнопку "Операцію з шаблонами", выберите шаблон свидетельства и операцию "Зберегти шаблон та відкрити для редагування". Все ключевые слова в шаблоне свидетельства находятся внутри специальных объектов типа "надпись". Чтобы переместить этот объект, кликните мышкой на объекте, появится граница объекта. Затем кликните мышкой на границе. Каретка изчезнет, объект готов для перемещения. Чтобы переместить объект "надпись", используйте клавиші "вверх", "вниз", "вправо", "влево" или перемещайте объект при помощи мышки. Для назначительного перемещения надписи (полтора-два милиметра) используйте комбинации клавиш:

    Ctrl+"влево"
    Ctrl+"вправо"
    Ctrl+"вверх"
    Ctrl+"вниз"



    Импортирование шаблонов.

    Функция импортирования шаблонов позволяет импортировать шаблон с компьютера, где шаблоны уже настроены и, таким образом, сэкономить время. Импортировать шаблон можно с любого источника в сети, который досупен по чтению. Для импротирования следует выбрать шаблон и какую-либо операцию с участием импортирования.
