Работа с ПК ЭРАГС в локальной компьютерной сети.

Если в отделе РАГС имеется несколько компьютеров, то для эффективной работы с ПК ЭРАГС их необходимо объединить в общую локальную компьютерную сеть. В качестве основного компьютера (т.е. такого, на котором будут расположены файлы БД ЭРАГС) следует выбрать наиболее производительный компьютер из имеющихся в отделе.

При установки ПК ЭРАГС на клиентский (т.н. не основной) компьютер во время установки следует указать "Другорядний", как показано на рисунке:



Для открытия файла БД ПК ЭРАГС на клиентском компьютере следует в файловом диалоге выбрать Сетевое Окружение, затем указать имя группы (на рисунке группа называется Rags), потом открыть основной компьютер (на рисунке MainPC), затем раскрыть папку с файлами базы данных и выбрать нужный файл.



Следует помнить, что на основном компьютере должен быть разрешен доступ по чтению и записи к папке, где находится файлы БД ПК ЭРАГС.
Для нормальной эксплуатации ПК ЭРАГС основной компьютер следует включать первым, а выключать последним.

Основная программа ПК ЭРАГС (Erags) в заголовке главного окна всегда пишет полный путь текущего файла базы данных. Если файл БД открыт с другого компьютера, то имени файлы предшествует имя компьютера, где этот файл находится или имя сетевого диска:



Программа Erags после первого старта всегда пытается открыть последний файл БД, с которым она работала. Файл не будет открыт, если нарушено сетевое соединение, переименован или удалён файл, переименован или не включен основной компьютер и т. п.
