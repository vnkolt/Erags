Шаблони документів

Програма Erags (основний компонент ПК ЕРАГС) для формування друкованих документів використовує спеціальні файли - шаблони документів, що по суті є макетами документів. Шаблон документа - файл у форматі rtf (Rich Text Format). З файлами цього формату можуть працювати багато різних програм, у тому числі Microsoft Word (WinWord).

Використання шаблонів дає можливість додати будь-якому документу, чи то актий запис, свідоцтво, довідка чи список, портрібний зовнішній вигляд. Насамперед це важливо для налаштування свідоцтва таким чином, щоб елементи даних точно влучали в відповідні графи бланка свідоцтва.

Налаштовувати шаблони можна за допомогою програми Microsoft Word (WinWord) чи будь-яким іншим редактором, що вміє працювати з rtf-файлами. У цій частині документації описані способи налаштування шаблонів за допомогою програми Microsoft Word 97. Для налаштування шаблону необхідно викликати відповідний файл шаблону.

Програмний комплекс ЕРАГС поставляється з усіма необхідними шаблонами, які готові для використання. Але може з'явитися необхідність перенастроїти шаблон (наприклад, у випадку зміни бланка свідоцтва) чи настроїти шаблон під конкретну модель принтера
