Организация работы с ПК ЭРАГС в областном отделе

Для эффективного использования ПК ЭРАГС в областном отделе РАЦС следует обеспечить сбор и пополнение файлов базы данных ПК ЭРАГС. Отделы РАГС в области могут передавать файлы БД областному отделу на различных носителях информации (компакт-дисках, магнито-оптических дисках, флэш-носителях и т.п.) или, если есть такая возможность с помощью электронной почты. Наиболее простым и дешёвым способом доставки файлов БД является использование флэш-носителей (так называемых "флэшек"). Флэш-носители миниатюрны, надёжны, удобны в использовании. Если БД ЭРАГС используется не только в отделах РАГС, но и в исполкомах сельских и поселковых советов, то можно организовать доставку данных, как показано на рисунке:

Другим способом пополнения базы данных является ввод алфавитных данных с помощью программы EraABC, входящей в состав ПК ЭРАГС.

Когда в областном отделе есть файлы БД ЭРАГС из отделов РАГС, появляется возможность использовать данные для выдачи повторных свидетельств, поиска данных, формирования списков и т.д. Чем полнее база данных, тем эффективнее она может быть использована. Например, на поиск каких-либо данных по запросу суда, прокуратуры, органов МВД и т.д. областной отдел может потратить несколько дней. Если же эти данные есть в базе данных областного отдела, то программа Erags найдёт эти данные всего за несколько секунд.
