Панели инструментов

Панели инструментов - это специальные окна с условными значками (пиктограммами). Панели инструментов выполняют ту же роль, что и меню, но несколько удобнее в использовании. ПК ЭРАГС имеет несколько панелей инструментов. Основные из них следующие:

Основная панель инструментов



Значок Быстрый вызов Комментарий
---- Открыть дерево файлов базы данных ЭРАГС.

Ctrl+O Открыть файл базы данных ЭРАГС. Файл БД ЭРАГС должен иметь тип (расширение) .er3.
---- Открыть файл базы данных ЭРАГС из списка недавно открываемых файлов ЭРАГС.

Ctrl+2 Выбрать регистрацию БРАКА.
Ctrl+3 Выбрать регистрацию РАСТОРЖЕНИЯ БРАКА.

Ctrl+U Открыть список географических названий.

Alt+Home Перейти к первой записи.
Alt+<- Перейти к предыдущей записи.
Alt+-> Перейти к следующей записи.
Alt+End Перейти к последней записи.
Alt+Insert Создать новую запись.
Alt+Delete Удалить текущую запись.
