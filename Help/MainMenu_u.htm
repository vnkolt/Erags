Головне меню

Головне меню - це смужка з текстом у верхній частині головного вікна програми. За допомогою елементів головного меню ви можете відкривати і створювати файли бази даних ЕРАГС, обрати потрібний вид реєстрації, здійснювати навігацію по базі даних і виконувати багато інших операцій.

При першому запуску програми Erags не всі елементи меню і панелей інструментів доступні. Це пов'язано з тим, що не створено жодного файлу бази даних.

Для початку роботи з ПК ЕРАГС потрібно створити файл бази даних. Для цього необхідно скористатися командою меню Файл/Новий або відповідною кнопкою на панелі інструментів.

Після вибору команди меню Файл/Новий з'явиться діалогове вікно для визначення імені файлу бази даних.



Кожен файл бази даних ЕРАГС є електронним аналогом книг реєстрації актових записів за відповідний рік. Тому, щоб запобігти плутанини та помилок, важливо, щоб ім'я файлу бази даних ЕРАГС починалося з чотирьох цифр - року, до якого відноситься цей файл.

Перед початком роботи варто перевірити установки і, якщо це необхідно, виправити значення установок. Для цього необхідно скористатися командою меню Огляд/Установи.

Після вибору команди меню Огляд/Установи з'явиться діалогове вікно Установи:

Зверніть увагу, що після назви населеного пункту потрібно вказати:

  • м. - для міста (місто);
  • с. - для села;
  • с-ще або слщ - для селища (селище);
  • смт - для селища міського типу (селище міського типу);
  • х - для хутора
Для визначення назви відділів РАГС необхідно використовувати скорочення:
  • ВРАГС - відділ реєстрації актів цивільного стану;
  • ВРАЦС - відділ реєстрації актів цивільного стану;
  • ОУЮ - обласного управління юстиції;
  • МУЮ - міського управління юстиції;
  • РУЮ - районного управління юстиції;
  • МРУЮ - міськрайонного управління юстиції;
  • ГУЮ - Головного управління юстиції
  • В - Виконком;
  • о - області;
  • р - району;
  • мр - міської ради;
  • ср - сільської ради;
  • сщр - селищної ради

Перед початком роботи з ПК ЕРАГС корисно також ознайомитися зі списками, що будуть використовуватися для швидкого введення даних. Щоб викликати списки для перегляду/редагування, необхідно скористатися командою меню Списки і відкрити потрібну таблицю.

Найбільш залежним до конкретного відділу РАГС є список медичних установ (Пологові будинки, лікарні, морги). Тому перед початком роботи варто відкрити цей список і визначити назви та псевдоніми тих медичних установ, що найбільше часто використовуються для реєстрації смертей і народжень у Вашому відділі РАГС.
