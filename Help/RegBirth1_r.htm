Элементы ввода при регистрации рождения

В нижеследующей таблице описаны элементы ввода при регистрации рождения. Номера элементов соответствуют номерам на рисунке.

№ Описание
1 Переключатель "Стать". Значение меняется с помощью клавиши Пробел или мыши. Значение для этого элемента нужно определить раньше других элементов.
2 Поле даты "Дата реєстрації". Значение по умолчанию - текущая дата.
3 Текстовое поле "Номер актового запису". Значение по умолчанию - следующий по порядку номер.
4 Текстовое поле "Прізвище новонародженого"
5 Поле со списком "Ім'я новонародженого".
6 Поле со списком "По батькові новонародженого".
7 Переключатель "Місце народження". Этот переключатель позволяет включать/отключить доступ для ввода непосредственно следующих за ним полей: "Місце народження: Держава", "Місце народження: Область", "Місце народження: Район", "Місце народження: Місто (село, селище)". По умолчанию значение єтих полей соответствует географической привязке отдела и они недоступні для ввода. Если нажать Enter или Tab (клавиши перехода на следующий элемент ввода, то текущитм элементом ввода станет поле 12 ("Дата народження"). Если же необходимо указать место рождения, отличное от географической привязки, то нужно воспользоваться этим переключателем и ввести данные поля 8, 9, 10 и 11.
8 Поле со списком "Місце народження: Держава". Значение по умолчанию - "Україна".
9 Поле со списком "Місце народження: Область". Значение по умолчанию соответствует географической привязке отдела.
10 Поле со списком "Місце народження: Район". Значение по умолчанию соответствует географической привязке отдела.
11 Поле со списком "Місце народження: Місто (село, селище)". Значение по умолчанию соответствует географической привязке отдела.
12 Поле даты "Дата народження". Значение по умолчанию - дата регистрации минус неделя.
13 Текстовое поле "Скільки народилося дітей". Можно вводить только цифры. Значение по умолчанию - "1".
14 Текстовое поле "Яка за рахунком дітина в матері". Можно вводить только цифры. Значение по умолчанию - "1".
15 Переключатель "Живонароджена чи мертвонароджена дитина". Значение по умолчанию - "живонароджена".
16 Текстовое поле "Номер медичного свідоцтва про народження".
17 Поле даты "Дата медичного свідоцтва про народження".
18 Поле со списком "Псевдонім медичної установи". Это поле служит для выстрого определения значения для следующего поля.
19 Поле со списком "Ким видано медичне свідоцтво про народження".
20 Переключатель "Підстава запису відомостей про батька". Значение по умолчанию "Актовий запис про шлюб".
21 Текстовое поле "Номер документу (підстави відомостей про батька).
22 Поле даты "Дата документу (підстави відомостей про батька)".
23 Поле со списком "Ким складено шлюб батьків". Это поле доступно только тогда, когда поле 20 ("Підстава запису відомостей про батька") имеет значение "Актовий запис про шлюб". Значение по умолчанию - сокращенное название Вашего отдела.
24 Поле со списком "Громадянство батька". Значение по умолчанию - "України".
25 Текстовое поле "Прізвище батька". Значение по умолчанию соответствует фамилии новорождённого.
26 Поле со списком "Ім'я батька". Значение по умолчанию соответствует имени новорождённого.
27 Поле со списком "По батькові батька".
28 Поле даты "Дата народження батька".
29 Поле со списком "Місце проживання батька: Держава". Значение по умолчанию - "Україна".
30 Поле со списком "Місце проживання батька: Область". Значение по умолчанию соответствует географической привязке отдела.
31 Поле со списком "Місце проживання батька: Район". Значение по умолчанию соответствует географической привязке отдела.
32 Поле со списком "Місце проживання батька: Місто (село, селище)". Значение по умолчанию соответствует географической привязке отдела.
33 Поле со списком "Місце проживання батька: Вулиця".
34 Текстовое поле "Місце проживання батька: Будинок".
35 Текстовое поле "Місце проживання: Квартира".
36 Поле со списком "Громадянство матері". Значение по умолчанию - "України".
37 Текстовое поле "Прізвище матері". Значение по умолчанию соответствует фамилии новорождённого.
38 Поле со списком "Ім'я матері".
39 Поле со списком "По батькові матері".
40 Поле даты "Дата народження матері".
41 Поле со списком "Місце проживання матері: Держава". Значение по умолчанию - "Україна".
42 Поле со списком "Місце проживання матері: Область". Значение по умолчанию соответствует географической привязке отдела.
43 Поле со списком "Місце проживання матері: Район". Значение по умолчанию соответствует географической привязке отдела.
44 Поле со списком "Місце проживання матері: Місто (село, селище)". Значение по умолчанию соответствует географической привязке отдела.
45 Поле со списком "Місце проживання матері: Вулиця". Значение по усолчанию соответсвует полю "Місце проживання батька: Вулиця" в том случае, если на обоих сторонах поля попарно идентичны значения полей 29 и 41 ("Держава"), 30 и 42 ("Область"), 31 и 43 ("Район"), 32 и 44 ("Місто").
46 Текстовое поле "Місце проживання матері: Будинок". Значение по умолчанию соответствует значению поля 34 ("Місце проживання батька: Будинок") в том случае, если на стороне отца и матери значения для поле 33 и 45 ("Вулиця") идентичны.
47 Текстовое поле "Місце проживання: Квартира". Значение по умолчанию соответствует значению поля 35 ("Місце проживання батька: Будинок") в том случае, если на стороне отца и матери значения для поле 34 и 46 ("Будинок") идентичны.
48 Переключатель "Заявник/Заявниця/Заявники". Этот переключатель служит для определения полей 49 и 50 ("ПІБ Заявника" и "Адреса заявника".
49 Текстовое поле "Прізвище, імя, по батькові заявника".
50 Текстовое поле "Адреса заявника".
51 Переключатель "Паспорт Заявника: тип". Переключатель может принимать следующие значения: "Паспорт", "Загальногромадянський Закардонний Паспорт", "Свідоцтво про Народження", "Тимчасова Посвідка на Проживання".
52 Текстовое поле "Паспорт Заявника: Серія".
53 Текстовое поле "Паспорт Заявника: Номер".
54 Текстовое поле "Паспорт Заявника: Дата видачі".
55 Поле со списком "Паспорт Заявника: Ким видано паспорт". Значение по умолчанию соответствует географической привязке отдела.
56 Текстовое поле "Серія свідоцтва про народження". Значение по умолчанию соответствует установке.
57 Текстовое поле "Номер свідоцтва про народження". Значение по умолчанию - следующий по порядку номер.
58 Текстовое поле "Помітки". Если поле пусто, то ЭРАГС формирует значение, исходя из введенных данных. Для заполнения поля удобно пользоваться контекстным меню (вызывается правой клавише мыши).
