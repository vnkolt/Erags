Элементы ввода при регистрации смерти

В нижеследующей таблице описаны элементы ввода при регистрации смерти. Номера элементов соответствуют номерам на рисунке.

№ Описание
1 Переключатель "Стать". Значение меняется с помощью клавиши Пробел или мыши. Значение для этого элемента нужно определить раньше других элементов.
2 Поле со списком "Громадянство". Значение по умолчанию - "України"
3 Поле даты "Дата реєстрації". Значение по умолчанию - текущая дата.
4 Текстовое поле "Номер актового запису". Значение по умолчанию - следующий по порядку номер.
5 Текстовое поле "Прізвище"
6 Поле со списком "Ім'я".
7 Поле со списком "По батькові".
8 Поле даты "Дата смерті". Значение по умолчанию - дата регистрации минус один день.
9 Возраст. При вводе каретка никогда не становится на поле "Возраст". Возраст вычисляется автоматически после ввода дат смерти и рождения.
10 Поле со списком "Місце смерті: Держава". Значение по умолчанию - "Україна".
11 Поле со списком "Місце смерті: Область". Значение по умолчанию соответствует географической привязке отдела.
12 Поле со списком "Місце смерті: Район". Значение по умолчанию соответствует географической привязке отдела.
13 Поле со списком "Місце смерті: Місто (село, селище)". Значение по умолчанию соответствует географической привязке отдела.
14 Поле со списком "Псевдонім причини смерті". Этот элемент предназначен для быстрого ввода причины смерти в следующее поле.
15 Текстовое поле "Причина смерті".
16 Поле со списком "Документ, що підтверджує факт смерті: Назва". Для ввода значения "Остаточне лікарське свідоцтво про смерть №" достаточно ввести "о", для "Попереднє лікарське свідоцтво про смерть №" - "п" и т.п.
17 Текстовое поле "Документ, що підтверджує факт смерті: Номер".
18 Поле даты "Дата документу, що підтверджує факт смерті". Значение по умолчанию соответствует дате регистрации.
19 Поле со списком "Псевдонім лікарської установи". Позволяет быстро определить значение для следующего поля.
20 Поле со списком "Ким видано медичне свідоцтво про смерть".
21 Поле даты "Дата народження".
22 Поле со списком "Місце народження: Держава". Значение по умолчанию - "Україна".
23 Поле со списком "Місце народження: Область". Значение по умолчанию соответствует географической привязке отдела.
24 Поле со списком "Місце народження: Район". Значение по умолчанию соответствует географической привязке отдела.
25 Поле со списком "Місце народження: Місто (село, селище)". Значение по умолчанию соответствует географической привязке отдела.
26 Поле со списком "Місце проживання: Держава". Значение по умолчанию - "Україна".
27 Поле со списком "Місце проживання: Область". Значение по умолчанию соответствует географической привязке отдела.
28 Поле со списком "Місце проживання: Район". Значение по умолчанию соответствует географической привязке отдела.
29 Поле со списком "Місце проживання: Місто (село, селище)". Значение по умолчанию соответствует географической привязке отдела.
30 Поле со списком "Місце проживання: Вулиця".
31 Текстовое поле "Місце проживання: Будинок".
32 Текстовое поле "Місце проживання: Квартира".
33 Текстовое поле "Ідентифікаційний Номер". Значение, введенное в єто поле используется для формирования списка для налоговой слжбы.
34 Переключатель "Паспорт померлого: тип". Переключатель может принимать следующие значения: "Паспорт", "загальногромадянський Закардонний Паспорт", "Свідоцтво про Народження", "Тимчасова Посвідка на Проживання".
35 Текстовое поле "Паспорт померлого: Серія".
36 Текстовое поле "Паспорт померлого: Номер".
37 Текстовое поле "Паспорт померлого: Дата видачі".
38 Поле со списком "Паспорт померлого: Ким видано паспорт". Значение по умолчанию соответствует географической привязке отдела.
39 Текстовое поле "Прізвище, ім'я, по батькові заявника". Автоматически предлагается такая же фамилия, как у умершего.
40 Текстовое поле "Адреса заявника". Автоматически предлагается значение, соответсвующее адресу умершего.
41 Переключатель "Паспорт Заявника: тип". Переключатель может принимать следующие значения: "Паспорт", "загальногромадянський Закардонний Паспорт", "Свідоцтво про Народження", "Тимчасова Посвідка на Проживання".
42 Текстовое поле "Паспорт Заявника: Серія".
43 Текстовое поле "Паспорт Заявника: Номер".
44 Текстовое поле "Паспорт Заявника: Дата видачі".
45 Поле со списком "Паспорт Заявника: Ким видано паспорт". Значение по умолчанию соответствует географической привязке отдела.
46 Текстовое поле "Серія свідоцтва про смерть". Значение по умолчанию соответствует установке.
47 Текстовое поле "Номер свідоцтва про смерть". Значение по умолчанию - следующий по порядку номер.
48 Текстовое поле "Помітки". Если поле пусто, то ЭРАГС формирует значение, исходя из введенных данных.
