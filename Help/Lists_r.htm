Списки

Программа Erags для ввода данных используют следующие списки:

  • географических названий
  • органов регистрации
  • гражданств
  • улиц
  • органов внутренних дел
  • родильных домов, больниц, моргов
  • финансовых учреждений
  • имён
  • отчеств
  • причин смерти

    • Списки служат для быстрого и удобного ввода данных. Списки географических названий и органов регистрации имеют древоподобную структуру, которая отображает логические связи соответствующих данных. Для работы с этими списками в программе Erags предусмотрены специальные окна, отображающие древоподобную структуру этих списков.

    Списки родильных домов, больниц, моргов и причин смерти имеют так называемые псевдонимы. Использование псевдонима позволяет ввести достаточно длинное название с помощью нескольких нажатий клавиш. Пример: для ввода причины смерти "Ішемічна хвороба серця" достаточно ввести псевдоним "іхс", а для ввода названия больницы "Донецьким областним лечебно-клиничним об'єдненням" достаточно ввести псевдоним "ДОЛКО".

    Список географических названий

    Список географических названий используется программой Erags там, где нужно указать географическое название. Список имеет древоподобную структуру. Порядок ввода географических названий всегда одинаков: страна, область, район, город (пгт, село). Рассмотрим ввод группы географических названий Україна, Вінницька, Барський, Діброва слщ. Название Україна будет введено автоматически, при переходе в поле "Область" будет введено название, указанное в настройках, список областей будет наполнен только областями Украины и для заполнения поля "Область" значением Вінницька достаточно ввести "ві". Когда поле "Держава" заполнено значением Україна, а поле "Область" - значением Вінницька, то список районов будет заполнен только районами Винницкой области и для ввода значения Барський в поле "Район" достаточно ввести лишь букву "Б". При этом список населённых пунктов будет заполнен только населёнными пунктами Барского района Винницкой области и для ввода значения Діброва слщ в поле "м.с.") достаточно лишь ввести букву "Д".

    Для того, чтобы просмотреть или изменить список географических названий нажмите комбинацию Ctrl+U или кнопку на главной панели инструментов. На рисунке ниже показано окно списка географических названий, с раскрытыми узлами списка.


    Навигацию по списку можно осуществлять с помощью мыши и клавишами управления курсором. Для редактирования списка воспользуйтесь контекстным меню. Для вызова контекстного меню нажмите правую клавишу мыши на каком-нибудь элементе списка. На рисунке ниже показано контекстное меню списка:

    Список органов регистрации

    Список органов регистрации используется программой Erags там, где нужно ввести орган регистрации, а также для печати конвертов. Список имеет древоподобную структуру. На рисунке ниже показано окно со список органов регистрации.


    Не все отделы из списка будут попадать в поля со списком. По умолчанию поля со списком будут наполняться только теми отделами РАГС, которые относятся к отделу, указанному в установках как вышестоящий отдел. Если Вы хотите, чтобы в поля со списком попадали отделы РАГС из другой области, воспользуйтесь контекстным меню. Для этого выберите какой-либо отдел, нажмите правую клавишу мыши и щёлкните на позиции "Додавати до поля зі списком":


    Список органов регистрации может служить также как справочник отделов РАГС, Вы можете просматривать и редактировать информацию об отделе: адрес, телефоны, примечания. Кроме того, используя список органов РАГС, Вы можете печатать конверты. Подробнее о печати конвертов смотрите в разделе Печать конвертов.

    Список гражданств

    Список гражданств используется программой Erags там, где нужно ввести гражданство. Список состоит из двух колонок: "Коротка Назва" и "Довга Назва". При вводе гражданства следует вводить короткое название. Для генерации документа Erags будет использовать длинное название. Пример: в поле "Громадянство" Вы вписали "Ізраілю", для генерации актовой записи в графу "Громадянство" Erags поместит "Держави Ізраіль". Для просмотра или редактирования списка гражданств следует выбрать его из меню "Списки". На картинке ниже показан список гражданств.


    Список улиц

    Список улиц используется программой Erags там, где нужно ввести улицу. Для удобства ввода рекомендуется в начале эксплуатации ПК ЭРАГС наполнить список улиц названиями, которые наиболее часто встречаются в Вашем отделе РАГС. Для просмотра или редактирования списка улиц следует выбрать его из меню "Списки". На картинке ниже показано окно с примером списка улиц.


    Список органов внутренних дел

    Список органов внутренних дел используется программой Erags там, где нужно ввести название органа, выдавшего паспорт. По умолчанию программа Erags предлагает тот орган внутренних дел, который указан в установках. Для просмотра или редактирования списка органов внутренних дел следует выбрать его из меню "Списки". На картинке ниже показано окно с примером списка органов внутренних дел.


    Список родильных домов, больниц, моргов

    Список родильных домов, больниц, моргов используется программой Erags при регистрации смерти (орган, выдавший медицинское свидетельство о смерти) и при регистрации рождения (орган, выдавший медицинское свидетельство о рождении). Для удобства ввода рекомендуется в начале эксплуатации ПК ЭРАГС наполнить этот список названиями соответствующих учреждений, которые наиболее часто встречаются в Вашем отделе РАГС. Этот список состоит из двух колонок "Псевдоним" и "Название". Вы можете определять псевдонимы по Вашему усмотрению, однако лучше, если псевдоним ассоциируется с названием. Использование псевдонимов позволяет вводить длинное название быстрее. Для просмотра или редактирования списка родильных домов, больниц и моргов следует выбрать его из меню "Списки". На картинке ниже показано окно с примером списка родильных домов, больниц и моргов.


    Список финансовых учреждений

    Список финансовых учреждений используется программой Erags при регистрации расторжения браков и перемены имени. Для удобства ввода рекомендуется в начале эксплуатации ПК ЭРАГС ввести несколько названий финансовый учреждений, которые наиболее часто встречаются в Вашем отделе РАГС, т.е. расположены недалеко от Вашего отдела РАГС. Обычно достаточно двух-трёх названий, чтобы обеспечить быстрый ввод названия финансового учреждения в соответствующее поле. Программа Erags позволяет ввести до 20-и названий финансовых учреждений. Для просмотра или редактирования списка финансовых учреждений следует выбрать его из меню "Списки". На картинке ниже показано окно для просмотра и редактирования c примером списка финансовых учреждений.


    Список имён

    Список имён используется программами Erags и EraABC всякий раз, когда нужно ввести имя в соответствующее поле. Каждое значение списка имён имеет признак "Пол". Чтобы изменить этот признак, нужно навести курсор мыши на соответствующую ячейку и щёлкнуть левой клавишей мыши. Для просмотра или редактирования списка имён следует выбрать его из меню "Списки". На картинке ниже показано окно для просмотра и редактирования списка имён.


    Список отчеств

Чтобы изменить этот признак, нужно навести курсор мыши на соответствующую ячейку и щёлкнуть левой клавишей мыши. Для просмотра или редактирования списка отчеств следует выбрать его из меню "Списки".     Список причин смерти

    Список причин смерти используется программой Erags для ввода причины смерти в регистрации смерти. Список состоит из двух колонок: "Псевдоним" и "Причина смерти". Использование псевдонимов позволяет быстро ввести причину смерти. Примеры: ввод псевдонима "аспмкпн" определит причину смерти "Атеросклеротичний порок мітрального клапану з перевагою недостатності.", а ввод псевдонима "іхсгкнн" определяет причину смерти "Ішемічна хвороба серця: гостра коронарна недостатність.". Перед эксплуатацией ПК ЭРАГС рекомендуется ознакомиться со списком причин смерти, чтобы в дальнейшем упростить его использование для ввода псевдонимов. Для просмотра или редактирования списка причин смерти следует выбрать его из меню "Списки". На картинке ниже показано окно со списком причин смерти.
