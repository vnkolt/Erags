Шаблоны документов

Программа Erags (основной компонент ПК "ЭРАГС") для формирования печатных документов использует специальные файлы - шаблоны документов, которые по сути являются макетами документов. Шаблон документа - файл в формате rtf (Rich Text Format). С файлами этого формата могут работать множество различных приложений, в том числе Microsoft Word (WinWord).

Использование шаблонов даёт возможность придать любому документу, будь то актовая запись, свидетельство, справка или список необходимый внешний вид. Прежде всего это важно для настройки свидельств таким образом, чтобы элементы данных точно попадали в соответствующие графы бланка свидетельства.

Настраивать шаблоны следует с помощью программы Microsoft Word (WinWord) или любым другим редактором, который умеет работать с rtf-файлами. В этой части документации описаны способы настройки шаблонов с помощью программы Microsoft Word 97. Для настройки шаблона необходимо вызвать соответствующий файл шаблона.

Программный комплекс ЭРАГС поставляется со всеми необходимыми шаблонами, которые уже настроены. Однако может появиться необходимость перенастроить шаблон (например, в случае изменения бланка свидетельства) или настроить шаблон под конкретную модель принтера.
