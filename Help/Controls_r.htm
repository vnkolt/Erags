Элементы ввода

Для ввода данных в ПК ЭРАГС используются следующие элементы ввода:

Элемент Внешний вид Комментарий
Текстовое поле Текстовое поле используется для ввода текстовых данных. Поля ввода номера актовой записи, фамилии, примечаний, номера дома, квартиры - это все текстовые поля. Некоторые текстовые поля автоматически изменяют текст при вводе. Например, при вводе серии паспорта буквы автоматически преобразуются в верхний регистр.
Поле даты

Поле даты используется везде, где требуется ввести дату. Дата регистрации, дата рождения, дата смерти, дата выдачи паспорта - это поля даты. Когда поле даты является текущим элементом ввода (белый фон) показаны только цифры и точки, например 27.07.2004. Если же текущий элемент ввода другой, то поле даты отображается иначе, например 29 липня 2004. При вводе даты следует вводить только цифры. Точки или другие разделители вводить не нужно. Если год вводимой даты больше, чем 1929 (установка операционной систему по умолчанию), то можно вводить две цифры года. В этом случае значения года 53, 64, 87 будут интерпретироваться соответственно как 1953, 1964 и 1987, а значение 03 будет интерпретироваться как 2003. Если же нужно ввести год до 1930, то следует вводить четыре цифры года.
Поле со списком

Поле со списком используется там, где можно выбрать значение из некоторого множества данных. Использование полей со списком позволяет значительно сократить время на ввод данных. Для некоторых данных при входе в поле со списком ЭРАГС автоматически предлагает некоторое значение, которое называется значением по умолчанию. Любое поле со списком можно раскрыть с помощью мыши или комбинации клавиш Alt+в. Однако, значение поля со списком подбирается и при обычном клавиатурном вводе. Например, если требуется ввести отчество "Іванівна" и такое значение есть в списке, то достаточно ввести одну или две первые буквы. Весьма эффективно поля со списком можно использовать для определения учреждений, выдавших медицинское свидетельство о смерти или о рождении, а также для определения причины смерти. В этих случаях можно использовать псевдонимы, что позволяет определить достаточно длинное название с помощью нажатия двух-трёх клавиш.

Переключатель

Переключатель используется там, где нужно выбрать значение из двух или более возможных, множество которых ограничено и не может быть изменено. Типичные примеры переключателей - поля для определения пола новорожденного или умершего, тип паспортного документа, основание записи сведений об отце. В некоторых случаях изменение значение переключателя приводит к изменению других элементов ввода.
Управляющая кнопка

Управляющие кнопки используются в различных диалоговых окнах и панелях инструментов. Управляющая кнопка сама по себе не меняет данных, она лишь позволяет вызвать диалоговое окно или меню, подтвердить ввод и т. п.
Флажок

Флажок используется для включения/выключения некоторого свойства или признака в некоторых диалоговых окнах. Пример - признак повторного свидетельства в диалоговом окне "Установи друку свідоцтва".


Элемент Внешний вид Комментарий
Группа переключателей

Группа переключателей используется в некоторых диалоговых окнах для выбора некоторого значения из совокупности допустимых значений. Пример - диалоговое окно "Шаблони Документів".
Список

Список служит для выбора некоторого значения из достаточно большой совокупности. Списки могут быть с одним или несколькими столбцами. Столбцы списков могут быть озаглавлены. Пример - список выбора шаблона документа в диалоговом окне "Шаблони Документів".
Древовидный список

Древовидный список служит для выбора некоторого значения из списка, имеющего древовидную структуру. Древовидный список имеет только один столбец. Элемент древовидного списка может быть снабжен небольшим изображением для ассоциативного восприятия. Примеры - формы "Список географічних назв", "Список органів реєстрації", диалоговое окно "Установи друку свідоцтва".
