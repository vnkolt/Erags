Импорт данных.

В программе Erags имеется возможность импортировать данные из файлов нового (.er3) или старого (.era) формата. Импорт позволяет решить следующие проблемы:

  • собрать в один файл БД нового формата данные из разных файлов как старого, так и нового формата;
  • импортировать данные из файла старого формата в файла нового формата.

    • Импорт может понадобиться в случае, когда программа Erags использоваласть в одном отделе РАГС на разных компьютерах, которые не были объединены в сеть и на каждом компьютере имеются свои файлы БД ЭРАГС. После того, как эти компьютеры были объединены в сеть, импорт поможет собрать данные на основном (т.е. главном) компьютере.

    Кроме того, импорт позволяет перенести данные из файлов старого формата (.era), которые были созданы с помощью программы Rags.

    Для осуществления операции импорта необходимо:

  • открыть (или создать) файл, в который Вы намереваетесь импортировать данные;
  • выбрать пункт "Імпортувати дані" и меню "Файл", как показано на рисунке;


  • появится диалоговое окно для импорта данных;


  • нажмите кнопку с тремя точками в верхней части окана, чтобы выбрать файл-источник;

  • после выбора файла-источника Вы можете изменить условия импорта и начать операцию импорта.


    • После завершения операции импорта программа Erags сообщит о результате операции.

    Импорт рекомендуется осуществлять на основном компьютере.
    Если Вы не знаете, где находятся файлы для импорта, Вы можете попробовать поискать их с помощью системного средства "Поиск". При этом можно указать файловую маску для поиска. Например, если указать маску 200*, то будут найдены (если они есть) файлы, имя которых начинаются со 200, т. е. файлы, имена которых начинаются с 2000, 2001, 2002 и т.д.
