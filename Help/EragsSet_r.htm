Состав комплекса ЭРАГС

В состав комплекса ЭРАГС входят компонеты:

  • программа Setup (утилита для установки комплекса на компьютер),
  • программа Uninstall (утилита для удаления ПК ЭРАГС с компьютера),
  • динамическая библиотека установки EraWiz (мастер установки),
  • программа для электронной регистрации Erags (основной компонент комплекса),
  • динамическая библиотека для импорта и экспорта данных ImpExp,
  • программа для просмотра и ввода алфавитных данных EraABC,
  • программа Era2Xml для конвертирования данных в xml-файлы для импорта программой АРМ "ДРАЦС",
  • файлы шаблонов документов и базы данных, файлы базы данных,
  • файлы встроенной электронной документации и электронного помощника.

    • По умолчанию (значение можно изменить во время установки) программа Setup устанавливает ПК ЭРАГС в папку C:\Erags. При этом внутри папки Erags создаются следующие папки:

  • DB - папка для файлов базы данных (создаётся только на основном компьютере)
  • Help - папка с файлами встроенной электронной документации и электронного помощника
  • Templates - папка с файлами шаблонов документов базы данных
  • ПК ЕРАГС - папка с файлами-ссылками для ярлыков ПК ЭРАГС на рабочем столе

    • См. также Установка ПК ЭРАГС
