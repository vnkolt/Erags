    • Общие сведения о программе Era2Xml


    Программа Era2Xml входит в состав ПК ЭРАГС и предназначена для конвертирования данных из файлов базы данных ПК ЭРАГС (файлов с расширением er3) в особые xml-файлы, которые затем можно импортировать в АРМ "ДРАЦС". Процедура конвертирования и последующий импорт данных в АРМ "ДРАЦС" позволяет избежать повторного ввода и сэкономит Вам время. Программа-инсталлятор ПК ЭРАГС создаст папку "ПК ЕРАГС" на рабочем столе, в которой есть ярлык для запуска программы Era2Xml. Для запуска программы Era2Xml необходимо кликнуть на этом ярлыке.


    Внешний вид программы Era2Xml

    Параметры конвертирования


    Перед конвертированием Вы можете установить следующие значения, которые будут учитываться при генерации выходных xml-файлов:

  • Флажок "Обрабатывать актовые записи о рождении" ("Оброблювати актові записи про народження")

    • Если флажок включён, то программа Era2Xml будет обрабатывать актовые записи о рождении и генерировать соответствующие xml-файлы. По умолчанию флажок включён.

  • Флажок "с" "по" ("з" "по") для данных о рождении

    • Если флажок включён, то появляется возможность определить временной период. К имени выходного файла будет добавляться указаный период. В выходные файлы попадут лишь те актовые записи о рождении, дата регистрации которых входит в указанный период.

  • Флажок "Обрабатывать актовые записи о смерти" ("Оброблювати актові записи про смерть")

    • Если флажок включён, то программа Era2Xml будет обрабатывать актовые записи о смерти и генерировать соответствующие xml-файлы. По умолчанию флажок включён.

  • Флажок "с" "по" ("з" "по") для данных о смерти

    • Если флажок включён, то появляется возможность определить временной период. К имени выходного файла будет добавляться указаный период. В выходные файлы попадут лишь те актовые записи о смерти, дата регистрации которых входит в указанный период.

  • Файловая папка для записи выходных xml-файлов.

    • По умолчанию предлагается папка с именем "Era2Xml" на том логическом диске, где расположена системная папка "Program Files". Чаще всего это логический диск "C:". Если Вы желаете изменить имя папки для записи выходных файлов, укажите её имя или воспользуйтесь кнопкой "..." для выбора папки из диалога.

  • Максимальное количество записей в выходном xml-файле.

    • По умолчанию предлагается значение 100. Вы можете указать любой значение больше 1. Не рекомендуется указывать слишком большие значения, так как выходной xml-файл может получиться слишком большим и в дальнейшем с ним будет тяжело работать. Если Вы используете Era2Xml впервые, установите максимальное значение записей 10 или 20, чтобы Вам не пришлось слишком долго обрабатывать полученные файлы в АРМ "ДРАЦС". Помните, что чем меньше будет это значение, тем больше выходных файлов получится.

  • Текстовые поля "Назва відділу РАЦС або виконкому", "Керівник", "Посада керівника", "Спеціаліст, що реєстрував народження", "Посада спеціаліста, що реєстрував народження", "Спеціаліст, що реєстрував смерті", "Посада спеціаліста, що реєстрував смерті" и соответствующие флажки.

    • По умолчанию все флажки "Залишити поле пустим" за исключением "Назва відділу РАЦС або виконкому", "Керівник" включены. Если Вы хотите, чтобы в выходной xml-файл попадали соответствующие данные, выключите флажки и определите значения полей.

    Конвертирование данных


    Для конвертирования данных Вам необходимо добавить один или более файлов базы данных ЭРАГС в список для конвертирования. Для этого кликните кнопку "Додати" и из файлового диалога выбирете файл, из которого Era2Xml будет выбирать данные для конвертирования.

    В список для конвертирования можно добавлять любое количество исходных файлов БД ЭРАГС, однако рекомендуется конвертировать по одному файлу.

    После появления в списке файлов для конвертирования хотя бы одного файла кнопка "Конвертувати" станет доступной и можно приступить непосредственно к конвертированию. Для этого следует нажать на кнопку "Конвертувати".

    Имена выходных файлов


    Программа Era2Xml формирует выходные файлы следующим образом:
    - для АЗ о рождении к имени исходного файла будет добавлено строка _BirthXXXX-YYYY.xml, где XXXX - порядковый номер первой записи, находящейся в выходном xml-файле, а YYYY - порядковый номер последней записи, находящейся в выходном xml-файле. Примеры.
    Пример 1.
    Исходный файл БД ЭРАГС имеет имя "2007 ВРАЦС Куйбишевського РУЮ м.Донецька.er3", максимальное число записей в выходном файле - 100, а в исходном файле есть 1037 записей о рождении. Тогда выходные xml-файлы с данными о рождении будут иметь такие имена:

    2007 ВРАЦС Куйбишевського РУЮ м.Донецька.er3_Birth0601-0700.xml

    Пример 2.
    Исходный файл БД ЭРАГС имеет имя "2007 ВРАЦС Куйбишевського РУЮ м.Донецька.er3", максимальное число записей в выходном файле - 100, указан период с 01.01.2007 по 31.01.2007, а в исходном файле есть менее 100 записей о рождении за этот период. Тогда выходной xml-файл с данными о рождении будет иметь такое имя:

    Импорт данных в АРМ "ДРАЦС"


    Чтобы импортировать данные, полученные с помощью программы Era2Xml, нажмите на кнопку "Імпорт" в АРМ "ДРАЦС".

    Откроется окно "Імпорт". В этом окне нажмите "Завантажити файл".

    В файловом диалоге выберите источник с xml-файлами, с помощью программы Era2Xml и откройте нужный файл.

    Если xml-файл содержит данные о рождениях, то список справа наполнится данными о рождении. Все импортированные записи будут иметь статус импорта "Необроблений". Для дальнейшей обработки дважды кликните на выбраной записи.

    Если xml-файл содержит данные о смертях, то список справа наполнится данными о смертях. Все импортированные записи будут иметь статус импорта "Необроблений". Для дальнейшей обработки дважды кликните на выбраной записи.

    Использование программы Era2Xml в сельских, поселковых и городских советах


    Если в сельском, поселковом или городском совете используется ПК ЭРАГС, то совет может выполнять конвертирование самостоятельно и передавать вышестояшему отделу РАГС на флеш-накопителе, другом носителе или по электронной почте только выходные xml-файлы. В этом случае специалистам отдела РАЦС не придётся вводить в АРМ "ДРАЦС" записи совета вручную, так как их можно будет импортировать из полученных xml-фалов.


    В настоящий момент в программе Era2Xml предусмотрено конвертирование только для двух видов регистрации - для рождений и смертей.
