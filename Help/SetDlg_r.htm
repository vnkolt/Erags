Установки

С помощью установок Вы можете изменить внутренние настройки программы Erags. Диалоговое окно "Установки" вызывается из главного меню программы Erags. Это диалоговое окно содержит восемь вкладок (разделов). Каждая вкладка предназначена для изменения установок определённого типа.

На рисунке ниже показан пример установок для вкладки "Загальні Установи".

Самые важные установки - географическое местоположение и названия органов. Следует обратить внимание, что в некоторых случаях требуется ввести одно и тоже название в разных падежах. При определении названия отдела РАГС ("Скорочена назва відділу" на рисунке) обязательно нужно использовать сокращения:

Вкладка "Народження", которая показана на рисунке ниже, позволяет менять свойства для регистрации рождения.


Вкладка "Одруження", которая показана на рисунке ниже, позволяет менять свойства для регистрации брака.


Вкладка "Розірвання Шлюбу", которая показана на рисунке ниже, позволяет менять свойства для регистрации расторжения брака.


Вкладка "Зміна", которая показана на рисунке ниже, позволяет менять свойства для регистрации перемены.


Вкладка "Смерть", которая показана на рисунке ниже, позволяет менять свойства для регистрации расторжения смерти.


Вкладка "Зовняшній вигдяд", которая показана на рисунке ниже, позволяет изменить фоновую раскраску окон программы Erags.


Вкладка "Архівування", которая показана на рисунке ниже, позволяет указать день недели, когда требуется выполнить резервное архивирование файлов базы данных, или отказаться от архивирования..



Если в Вашем отделе есть несколько компьютеров и они объединены в локальную сеть, то программа Erags будет выполнять автоматическое архивирование данных только на основном компьютере, т.е. на том компьютере, где физически находятся файлы базы данных ПК ЭРАГС. Поэтому установки архививирования действительны только для основного компьютера.


Вкладка "Офіс", которая показана на рисунке ниже, позволяет переключить программу Erags на интеграцию с пакетом OpenOffice.org или с Microsoft Office. Если в Вашем отделе установлена нелегальная копия пакета Microsoft Office, рекомендуется включить интеграцию с OpenOffice.org и деинсталлировать Microsoft Office. Фактически, пакет OpenOffice.org обладает теми же возможностями, что и Microsoft Office, но, в отличие от Microsoft Office, является бесплатным офисным пакетом, поэтому Вы можете использовать его на законных основаниях. Загрузить последнюю версию пакета OpenOffice.org можно с сайта http://openoffice.org. Начиная с сентября 2008-го года установить пакет OpenOffice.org можно непосредственно с установочного диска ПК ЭРАГС.




Вкладка "Штампи", которая показана на рисунке ниже, позволяет включить или выключить использование штампов для генерации документов. Для использования штампов предусмотрены специальные ключевые слова !Штамп1! и !Штамп2!. Если в шаблоне документа присутствует какое-либо из этих ключевых слов, и использование штампа/штампов включено, то Erags при генерации документа будет помещать вместо ключевого слова текст, соответсвующий этому штампу.
