Правила ввода данных в поле "Будинок" для и соответсвие значения поля "Будинок" ключевым словам "Буд", "Б" и "К":
- вместо ключевого слова "Буд" всегда помещаются данные, которые были введены в поле "Будинок";
- вместо ключевого слова "Б" всегда часть данных, которые соответствуют номеру дома в поле "Будинок";
- вместо ключевого слова "К" всегда часть данных, которые соответствуют номеру корпуса (блока, секции и т.п.) в поле "Будинок";
- в случае, если в адресе присутствует номер блока, секции и т.п., то в при вводе данных в поле "Будинок" необходимо использовать разделители "," "/" и "\";
- при вводе данных в поле "Будинок" можно импользовать следующие сокращения: блок, бл, б-к (блок); корпус, кор, крп, к-c (корпус); секція, сек, скт, с-я (секция).
