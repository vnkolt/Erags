Панелі інструментів

Панелі інструментів - це спеціальні вікна з умовними значками (піктограмами). Панелі інструментів виконують ту ж роль, що і меню, але трохи зручніше у використанні. ПК ЕРАГС має кілька панелей інструментів. Основні з них наступні:

Основна панель інструментів



Відкрити дерево файлів бази даних ЕРАГС.
Створити новий файл бази даних ЕРАГС.
Відкрити файл бази даних ЕРАГС. Файл БД ЕРАГС повинний мати тип (розширення) .er3.
Відкрити файл бази даних ЕРАГС зі списку файлів, що недавно відкривалися.

Обрати реєстрацію РОЗІРВАННЯ ШЛЮБУ.

Обрати реєстрацію СМЕРТІ.
Викликати діалогове вікно "Зміни та доповнення". Застосовується тільки при реєстрації народження для визначення прізвища, імені та по батькові новонародженого у випадку реєстрації за заявою про визнання батьківства. Можна також визначити нове прізвище матері.
