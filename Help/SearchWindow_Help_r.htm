    • Кнопка "Новый поиск"



    Нажмите эту кнопку, чтобы очистить список результатов поиска (окно справа) и начать поиск с другими заданиями.




    Кнопка "Записать результаты поиск"



    Нажмите эту кнопку, чтобы записать результат поиска в .csv-файл для дальнейшей работи с ним с помощью программы Exel.




    Кнопка "Маршрут поиска"



    Нажмите эту кнопку, чтобы определить маршрут (файловую папку) поиска. По умолчанию ЭРАГС будет искать в папке, где находится файл БД ЭРАГС, который открывался в последним.




    Кнопка "Начать поиск"



    Нажмите эту кнопку, чтобы начать поиск. Список заданий (слева) должен содержать хотя бы одно задание (фамилию, имя, или отчество) для поиска.




    Кнопка "Полное совпадение"



    Нажмите эту кнопку, чтобы установить режим полного совпадения. Если этот режим установлен, в список результатов попадут только актовые записи, которые полностью удовлетворяют какому-либо заданию поиска. Пример: в списке заданий указана фамилия "Ткач". В список результатов попадут (если они существуют в БД) такие записи: Ткач Іван Петрович Ткач Ольга Леонідівна Ткач Оксана Олександрівна




    Кнопка "Частичное совпадение"



    Нажмите эту кнопку, чтобы установить режим частичного совпадения. Если этот режим установлен, в список результатов попадут только актовые записи, которые частично удовлетворяют какому-либо заданию поиска. Пример: в списке заданий указана фамилия "Ткач". В список результатов попадут (если они существуют в БД) такие записи: Ткач Іван Петрович Ткаченко Ольга Леонідівна Ткачова Оксана Олександрівна Білогуров-Ткачук Володимир Павлович




    Кнопка "Поиск в АЗ о рождении"



    Нажмите эту кнопку, чтобы включить/выключить поиск актовых записей о рождении.




    Кнопка "Поиск в АЗ о браке"



    Нажмите эту кнопку, чтобы включить/выключить поиск актовых записей о браке.




    Кнопка "Поиск в АЗ о расторжении брака"



    Нажмите эту кнопку, чтобы включить/выключить поиск актовых записей о расторжении брака.




    Кнопка "Поиск в АЗ о перемене"



    Нажмите эту кнопку, чтобы включить/выключить поиск актовых записей о перемене.




    Кнопка "Поиск в АЗ о смерти"



    Нажмите эту кнопку, чтобы включить/выключить поиск актовые записи о смерти.




    Кнопка "Открыть найденную АЗ"



    Нажмите эту кнопку, чтобы открыть найденную актовую запись. В списке результатов (окно справа) необходимо указать запись, которую слудует открыть.




    Список заданий поиска



    Перед поиском слудует определить хотя бы одно (но не более 99) задание. Каждое задание состоит из трёх частей: фамилия, имя, отчество. ЭРАГС будет искать по указанному заданию, если хотя бы одна часть задания определена, т.е. непуста. Если определены две или три части задания, ЭРАГС искать такие АЗ, которые удовлетворяют всем указанным частям условия. Пример: В списке заданий определены три задания: 1 ВолковОлександр 2 ОльгаГен 3 олектрофан В режиме частичного совпадения в список результатов попадут (если они есть): Волков Олександр Віталійович Волкова Олександра Миколаївна Кузьмук Ольга Геннадіївна Заблоцька Ольга Євгенівна Біла Олександра Митрофанівна Дядюк Олексій Митрофанович Максименко Олександр Митрофанович




    Список найденных АЗ



    В этот список попадают АЗ, найденные в результате поиска.
    Любой найденный АЗ можно сразу же открыть.
