Вимоги до апаратного забезпечення

Для експлуатації програмного комплексу ЕРАГС потрібно комп'ютер Pentium (чи сумісний з ним) з тактовою частотою процесора не менш 133 MHz (рекомендується 633 MHz чи більш), пам'яттю 32Mb (рекомендується 128 Mb чи більш). Рекомендується (але необов'язково), щоб до комп'ютера був підключений маніпулятор типу "миша" чи аналогічний пристрій позіціювання курсору. Для повноцінної роботи з ПК ЕРАГС бажано, щоб монітор і відеоадаптер підтримували повнокольоровий (TrueColor) екранний режим не менш 800x600 пікселів

Програма Erags дозволяє друкувати документи на принтери будь-яких типів, але для ефективної роботи варто використовувати лазерний принтер. Більшість сучасних моделей лазерних принтерів мають високу швидкість і гарну якість друку.

Найбільш зручними принтерами для експлуатації у відділах РАГС є високошвидкісні сучасні лазерні принтери з верхнім лотком для подачі паперу, як, наприклад, принтер Canon LASER SHOT LBP-1120 чи аналогічний.

Якщо у відділі РАГС є в наявності два комп'ютери, але лише один принтер, є сенс використовувати принтер як загальний для обох комп'ютерів. Для цього необхідно зв'язати обидва комп'ютери в локальну мережу і відкрити доступ до принтера по мережі. Для підключення комп'ютера до локальної мережі необхідно, щоб він мав мережний адаптер.

Вимоги до програмного забезпечення

На комп'ютері повинна бути встановлена будь-яка з операційних систем Windows 98, Windows Me, Windows NT 4.0, Windows 2000, Windows XP чи Windows 7. Операційна система повинна бути російськомовної (україномовної) чи русифікованої (українізованої), тобто підтримувати кириличні шрифти і клавіатурне введення українською мовою. На комп'ютері повинна бути також установлена програма WinWord з пакета Microsoft Office 97 (Microsoft Office 2000, Microsoft Office XP, Microsoft Office 2003) або OpenOffice.org.
