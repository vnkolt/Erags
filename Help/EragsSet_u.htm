Склад програмного комплексу ЕРАГС

До складу комплексу ЕРАГС входить наступні компоненти

  • програма Setup (утиліта для установки комплексу на комп'ютер),
  • програма Uninstall (утиліта для видалення ПК ЕРАГС с комп'ютеру),
  • дінамічна бібліотека установки EraWiz (Майстер установки),
  • програма для електронної реєстрації Erags (основний компонент комплексу),
  • дінамічна бібліотека для імпорту та експорту даних ImpExp,
  • програма для перешляду і вводу алфавітных даних EraABC,
  • програма Era2Xml для конвертувння даних в xml-файли для імпорту програмою АРМ "ДРАЦС",
  • файли шаблонів документів и бази даних, файли бази даних,
  • файли вбудованної електронної документації і електронного помічника.

    • За умовчаванням (значення можна змінити під час установки) програма Setup установлює ПК ЕРАГС у папку C:\Erags. При цьому усередині папки Erags створюються наступні папки:

  • DB - папка для файлів бази даних (створюється тільки на основному комп'ютері)
  • Help - папка з файлами вбудованої електронної документації й електронного помічника
  • Templates - папка з файлами шаблонів документів і бази даних
  • ПК ЕРАГС - папка з файлами-посиланнями для ярликів ПК ЕРАГС на робочому столі
