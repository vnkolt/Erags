    • Відкрити файл бази даних



    Натисніть цю кнопку, щоб відкрити файл бази даних.
    Файл бази даних ЕРАГС має тип .er3
    Якщо файл бази даних знаходиться на іншому комп'ютері в локальній комп'ютерній мережі,
    у файловому діалозі треба обрати
    \\ім'я комп'ютеру\папка ЕРАГС\файли БД ЕРАГС\ім'я файлу.er3,
    наприклад:
    - у файловому діалозі обрати Сетевое Окружение;
    - відкрити компьютер, де знаходиться БД ЕРАГС, наприклад DBServer;
    - відкрити папку, де знаходиться БД ЕРАГС, наприклад Erags\DB;
    - відкрити файл БД ЕРАГС, наприклад 2004 Горлiвський мiский ВРАЦС Донецького ОУЮ.er3
    Повне ім'я файлу БД у данному випадку:
    \\DBServer\Erags\DB\2004 Горлівський міский ВРАЦС Донецького ОУЮ.er3
